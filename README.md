## SGPA Calculator for FAST-NUCES

A vibrant, playful, and user-friendly SGPA (Semester Grade Point Average) calculator tailored specifically for FAST-NUCES grading criteria. Built with a colorful aesthetic, interactive buttons, and responsive backgrounds, this tool makes the stress of calculating your GPA a lot more delightful!

**Live Demo:** 
[Check out the live calculator on Netlify here!](https://fast-cute-gpa-calculator.netlify.app)


[Or the GitHub link here!](https://dania-mustafa.github.io/fast-cute-gpa-calculator/)

## Features

### Core Functionality
* **Accurate FAST Grading:** Pre-configured with standard credit hours and university grading scales.
* **Add/Remove Courses:** Flexibility to calculate your GPA for as few or as many courses as you took this semester.
* **Focused SGPA BOX:** The bottom box is made to concentrically display the SGPA calculated by the button.

### The Aesthetics & UI
* **Vibrant Color Palette:** To exhibit a lively, high-contrast, and energetic color scheme.
* **Playful Interactive Buttons:** Custom-designed buttons with hover effects and color gradients for adding courses, removing courses, and calculating the SGPA.
* **Dynamic Background Images:** The vibrant background adds a clean and fresh look to the calculator.
* **Full Responsiveness:** Works flawlessly on desktops, tablets, and is pleasant on phones.

## Built With

* **HTML5** - For the semantic structure.
* **CSS3** - For the vibrant colors, animations, buttons, and custom backgrounds.
* **JavaScript (ES6+)** - For the calculation logic and dynamic DOM manipulation.

Simply select your expected or obtained letter grade and the credit hours of the course, and the JavaScript logic handles the rest in real-time.
